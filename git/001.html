    git常用命令

    xiaoqiang 4/6/2021 git使用

    1.git配置

    git config --global user.name ""
git config --global user.email ""
    1
    2

    2.查看全局配置

    git config –list –global
    1

    3.查看指定配置

    git config user.name
git config user.email
    1
    2

    4.git初始化

    git init
    1

    5.查看状态

    git status || git status -s || git status -short
    1

    6.工作区到暂存区

    git add *
git commit - m ""
    1
    2

    7.从暂存区恢复文件到工作区

    git checkout 文件名  (慎用，有风险！！！)
    1

    8.查看工作区修改的代码

    git diff
    1

    9.查看已经提交到暂存区的历史版本

    git log   ||   git reflog
    1

    10.恢复文件到指定版本

    git reset --hard 版本号
    1

    11.生成ssh密匙

    ssh-keygen -t rsa -C "邮箱"
    1

    12.将暂存区的上传到github仓库

    git remote add origin 仓库地址
git push -u origin master
    1
    2

    13.从远程仓库克隆到本地

    git clone 远程仓库地址
    1

    14.推、拉代码

    git pull 拉
git push 推
git push -f  强制推送
    1
    2
    3

    15.创建子分支

    git checkout -b 分支名
    1

    16.查看分支

    git branch    查看本地分支
git branch -a  查看远程分支
git branch 分支名称 -d  删除本地分支
git branch 分支名称 -D  强制删除本地分支
    1
    2
    3
